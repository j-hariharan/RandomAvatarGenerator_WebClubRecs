RoboAvatar

Randomly generate cute robot avatars!

Get Started

RoboAvatar API is very simple to use.

Base URL

http://localhost:3000/

API Endpoint

/generate

Query Params:

  • seed - the seed used to generate the pseudo-random avatar. The same seed always produces the same avatar! Omitting this param will give a random avatar each time.
  • type - specifies the image format to be generated. "png" and "svg" are the supported values. Omitting this defaults to svg outputs.
